# Flutter SDK Intrallation Steps for windows

## Step-1

Download Git from this link - https://github.com/git-for-windows/git/releases/download/v2.52.0.windows.1/Git-2.52.0-64-bit.exe

## Step-2

Download SDK Zip file - https://docs.flutter.dev/install/manual#install-flutter

## Step-3

- Check for free space in your drive and decide one drive to install the sdk.
- Extract the `SDK Zip` in that decided drive/space.

## Step - 4

You have to setup path.

- Go to SDK file that you extracted, then go to bin folder and copy the path address.
- Press `Window Key` and type `Edit the system environment variables` and press enter.
- Click on `Environment Variables`
- Under User Variables look for path.
- Double click on path
- Click new and paste your copied sdk bin address.

## Step - 5

- Close all your terminals
- Reopen a new terminal then type `flutter doctor`
